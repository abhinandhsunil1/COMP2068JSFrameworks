Class chat app 
A simple realtime chat application built Node.js, Express, and socket.io.
This app allows students in class to chat to each other live.

Features
- Real time messageing 
- user can join with their own name
- Shows when people enter chat
- Messgae can be seen instantly
- easy to use

Tech used to make the software
- Node.js
- Express.js
- Socket.io
- Html
- Css
- Javascript
- Nodemon

How it works
1 Server setup
- Express server static files
- Socket.io handles realtime communication
- shows when user joins
- user sent message
- users disconnects

2 Client side
- Connect to server using Socket.io
- Sent message when user click send
- Recieves new messages instantly
- Add people to chat window when they enter

To get access
Students will have to:
1 clone the repo
2 install dependancies
3 run the app
4 enter their name
5 join the live chat
